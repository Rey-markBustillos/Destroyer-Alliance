Implementation Notes

Bitmask layout (8 directions):
  128  1  2
   64     4
   32 16  8

Bits: 1=top,2=top-right,4=right,8=bottom-right,16=bottom,32=bottom-left,64=left,128=top-left
getBitmask checks 8 neighbours and returns 0-255 mask.
The demo renders a tileset where each index corresponds to the mask.
You can replace the tileset lookup with your sprite coordinates.
